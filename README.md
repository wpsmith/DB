Interacting with a custom DB table within WordPress.

## Requirements

*   PHP 7.2.0 or above
*   WordPress 4.4 or higher (for transient keys longer than 64 characters).

## Installation

The best way to use this package is through Composer:

```BASH
$ composer require wpsmith/DB
```
